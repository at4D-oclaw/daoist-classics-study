# 道可道，非常道 —— 十部经典，90天入门

```
道之为物，惟恍惟惚。
惚兮恍兮，其中有象；
恍兮惚兮，其中有物。
                ——《道德经》第二十一章
```

两千五百年前，老子写下五千言，转身西去，留下中国哲学史上最深邃的谜题。此后，庄子以寓言解之，《清静经》以修身验之，《阴符经》以天道证之，历代道者以生命践之……

**这个项目要回答一个问题**：一个现代人，如何在90-180天内系统走进道教经典的世界？

不是碎片化地刷"道德经金句"，不是功利化地学"道家智慧管理术"，而是——用教育科学的方法，从根基到殿堂，一步步走完这条路。

## 方法论：不是"读完"，而是"转化"

### SOLO分类理论

借用教育心理学中的**SOLO分类理论**，为每部经典设计五层认知阶梯：

```
前结构 ──→ 单点结构 ──→ 多点结构 ──→ 关联结构 ──→ 抽象拓展
 (听过)    (能说一个)   (能列多个)   (能连起来)   (能用出去)
```

每层都有明确的学习目标和自评标准。你始终知道自己在哪里，下一步去哪里。

### 项目式学习（PBL）

每读完一部经典，你必须交出**作品**——

- 一篇与道友交流的**学习体会**（800-1500字）
- 一份**思维导图或知识图谱**
- 一段**实践记录**（静坐日志、抄经、21天生活实验……）
- 可选：朗诵录音、书法、短视频

> 读经不是目的，转化才是。不产出作品的学习，只是信息过客。

## 常见问题

**Q：90天真的能学完10部经典吗？**
A：90天是冲刺模式（每天3小时+），适合假期集中学习。科学规划的从容节奏是140-160天（每天1.5-2小时）。速成不等于降低标准——SOLO五层框架和作品产出要求不变，只是压缩重复练习、提高学习密度。

**Q：我没有任何基础，能看懂吗？**
A：整套体系就是为零基础设计的。第一阶段从"听过但不懂"开始，每一步都有具体的任务指引。难度曲线呈波浪式上升，不会突然面对无法理解的文本。

**Q：必须按顺序学吗？**
A：建议按顺序。五阶段有内在的递进逻辑——先建立哲学根基，再进入修身实践。但道德经和庄子之间可以灵活安排。

**Q：这些经典有白话文版本吗？**
A：每部经典的推荐资源中都标注了⭐入门首选的注释本，如陈鼓应《老子今注今译》《庄子今注今译》。
