# 第二阶段：修身入门 —— 清静经 + 阴符经

> 从宇宙论走向身心修炼的桥梁（约3周 · 清静经391字 + 阴符经~340字）

---

## 第三章：清静经

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《太上老君说常清静经》 |
| **作者** | 传为太上老君（老子）所说，实为后世道徒所编 |
| **年代** | 约成书于唐代或更早（有王玄览等唐人注本传世） |
| **篇幅** | 约400字（含后人增补部分约600字） |
| **核心思想** | 清静为本、遣欲澄心、内观自省、悟道修真 |
| **历史地位** | 道教日常诵持的经典之一，修身入门的首选 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能听过"清静"二字，但不知道它是一部经典的核心；对道教修身没有概念 |
| **单点结构** | 识别一个核心概念 | 能解释"清静"的含义——心无杂念、不被欲望牵引；或能理解"遣欲"的基本意思 |
| **多点结构** | 列举多个关键概念 | 能说出清静、遣欲、澄心、内观、真常、道、德、妄心、真心等核心概念 |
| **关联结构** | 理解概念间的内在联系 | 能理解"遣欲"是达到"清静"的手段，"清静"是悟"道"的前提；能将"内观其心，心无其心"与修行实践联系 |
| **抽象拓展** | 迁移应用与创新思考 | 能将"清静"智慧应用于应对信息过载时代的焦虑；能比较清静经的"遣欲"与佛教的"断烦恼"；能设计个人的清静修行方案 |

### 三、PBL驱动问题

**核心驱动问题**：在信息爆炸、欲望泛滥的现代社会，"清静"还是一种可能的状态吗？我们如何从400字的《清静经》中找到内心的安宁？

**子问题链**：
1. 为什么道教把"清静"看得如此重要？
2. "遣欲"是不是意味着要消灭所有欲望？
3. "内观其心，心无其心"是什么意思？如何做到？
4. 清静经与道德经的"无为"有什么关系？

### 四、分阶段学习任务

#### 阶段一：诵读入门与义理研习（第1周）—— 前结构→多点结构

**学习内容**：
- 反复诵读清静经全文（建议每日诵读1-3遍），能背诵全文
- 阅读清静经注释，整理核心概念
- 理解清静经的修行逻辑

**任务清单**：
- [ ] 能背诵清静经全文
- [ ] 制作"清静经核心概念表"（至少8个概念）
- [ ] 画出清静经的修行逻辑图（遣欲→澄心→清静→悟道）

**作品产出**：
- 抄经作品：工整抄写清静经全文（硬笔或毛笔皆可）
- 思维导图：《清静经修行路径图》
- 学习体会：《清静经的核心密码》（800-1500字）

#### 阶段二：静坐体验与清静生活（第2周）—— 关联结构→抽象拓展

**学习内容**：
- 尝试以清静经为指导的静坐练习，将经文义理与静坐体验对照
- 将"清静"智慧应用于日常生活
- 理解"清静"与道德经"无为"的关系

**任务清单**：
- [ ] 每日静坐15-30分钟，以清静经为心法
- [ ] 设计"21天清静生活实验"方案并执行
- [ ] 写出清静经与道德经的思想关联分析（300字）

**作品产出**：
- 静坐日志：《7天清静经静坐记录》
- 关联分析：《清静经与道德经的对话》
- 实践报告：《清静生活实验心得》（1000字）

### 五、推荐参考资源

## 第四章：阴符经

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《黄帝阴符经》 |
| **作者** | 传为黄帝所著，实为唐代以前道流所编 |
| **年代** | 约成书于南北朝至唐代 |
| **篇幅** | 约300-400字（分三章：上篇神仙抱一演道章、中篇富国安人演法章、下篇强兵战胜演术章） |
| **核心思想** | 天人合一、观天之道、盗机应时、知之修炼 |
| **历史地位** | 与《道德经》《南华经》并称道家三经，兵家、纵横家亦重之 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能从未听说过此经；对"天人合一"只有模糊印象 |
| **单点结构** | 识别一个核心概念 | 能解释"观天之道，执天之行"的基本含义——观察自然规律并遵循它 |
| **多点结构** | 列举多个关键概念 | 能说出天道、盗机、时动、三才（天地人）、五贼、修炼等核心概念 |
| **关联结构** | 理解概念间的内在联系 | 能理解"盗机"不是偷盗而是把握时机；能将"天人合一"与具体的修炼实践联系 |
| **抽象拓展** | 迁移应用与创新思考 | 能将"观天之道"应用于现代决策和战略思考；能比较阴符经与兵法思想的异同；能从"盗机"角度分析成功案例 |

### 三、PBL驱动问题

**核心驱动问题**：古人如何通过观察天地运行的规律，来指导人的行为与修炼？这种"天人合一"的智慧对现代人有什么启示？

**子问题链**：
1. "观天之道，执天之行，尽矣"——这句话为什么说"尽矣"？
2. "五贼"指的是什么？为什么用"贼"这个字？
3. "天人合发，万化定基"——天人如何"合发"？
4. 阴符经为什么受到兵家和纵横家的重视？

### 四、分阶段学习任务

#### 阶段一：初读感知（第1周）—— 前结构→单点结构

**学习内容**：
- 诵读阴符经全文，逐句理解字面意思
- 了解阴符经的来历与历史地位

**任务清单**：
- [ ] 诵读阴符经全文至少10遍
- [ ] 写出你对"观天之道，执天之行"的初步理解（200字）

**作品产出**：
- 抄经作品：工整抄写阴符经全文
- 初读笔记：《阴符经初印象》

#### 阶段二：义理深研（第2周）—— 单点结构→多点结构

**学习内容**：
- 阅读阴符经注释（推荐张果、李筌注）
- 理解上中下三篇的不同侧重，整理核心概念

**任务清单**：
- [ ] 阅读至少2个注释版本
- [ ] 制作"阴符经核心概念表"（至少8个概念）

**作品产出**：
- 思维导图：《阴符经思想结构图》
- 学习体会：《阴符经的"天人密码"》（800-1500字）

#### 阶段三：关联理解与实践应用（第3周）—— 多点结构→抽象拓展

**学习内容**：
- 将阴符经与道德经、清静经进行比较
- 将"观天之道"的智慧应用于日常生活决策
- 探索阴符经在现代管理、决策中的应用

**任务清单**：
- [ ] 写出阴符经与道德经的思想关联分析（500字）
- [ ] 用阴符经的"盗机"思想分析一个历史或现实案例
- [ ] 进行7天"观天之道"实践：每天记录一个自然现象并思考其启示

**作品产出**：
- 比较分析：《道德经与阴符经的对话》
- 实践记录：《7天"观天之道"日记》
- 交流文章：《阴符经给现代人的启示》（1000字）
