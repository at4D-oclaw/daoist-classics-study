## 第八章：龙门心法

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《龙门心法》 |
| **作者** | 王常月（全真龙门派第七代律师） |
| **年代** | 清康熙年间（约17世纪后半叶） |
| **篇幅** | 约2万字（分上下两卷，共24篇） |
| **核心思想** | 持戒修心、修行次第、性命双修、真功真行 |
| **历史地位** | 全真龙门派的核心典籍，道教修行次第的系统阐述 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能听过"全真教""龙门派"等名称，但不知道其修行体系；对"修行次第"没有概念 |
| **单点结构** | 识别一个核心概念 | 能解释"持戒"的基本含义——遵守戒律是修行的基础；或能说出"性命双修"的基本意思 |
| **多点结构** | 列举多个关键概念 | 能说出持戒、修心、性命双修、真功真行、修行次第、龙门心法等核心概念 |
| **关联结构** | 理解概念间的内在联系 | 能理解"持戒"是"修心"的基础，"修心"是"性命双修"的前提；能将龙门心法的修行次第与前面学过的经典联系 |
| **抽象拓展** | 迁移应用与创新思考 | 能理解道教修行的系统性与渐进性；能将"持戒修心"应用于现代人的自律修行；能设计个人的修行次第 |

### 三、PBL驱动问题

**核心驱动问题**：道教修行有没有一条"正确的道路"？龙门心法的修行次第对我们规划个人成长有什么启发？

**子问题链**：
1. 为什么王常月如此强调"持戒"？
2. "性命双修"中的"性"和"命"分别指什么？
3. 道教的"修行次第"与佛教的"修行阶位"有什么异同？
4. "真功真行"——什么是道教修行中的"真"？

## 第九章：坐忘论

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《坐忘论》 |
| **作者** | 司马承祯（唐代著名道士，茅山宗第十二代宗师） |
| **年代** | 唐代（约8世纪） |
| **篇幅** | 约3000字（分七篇） |
| **核心思想** | 坐忘修心七阶：敬信、断缘、收心、简事、真观、泰定、得道 |
| **历史地位** | 道教静坐修心的经典之作，融合道、释、儒三家思想 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能听过"坐忘"一词，但不知道它有系统的七阶修行法；对"静坐"只有模糊印象 |
| **单点结构** | 识别一个核心概念 | 能解释"坐忘"的基本含义——通过静坐忘却外物、回归本心；或能说出七阶中的一阶 |
| **多点结构** | 列举多个关键概念 | 能说出敬信、断缘、收心、简事、真观、泰定、得道七阶的内容 |
| **关联结构** | 理解概念间的内在联系 | 能理解七阶之间的递进关系；能将"坐忘论"与清静经、道德经的思想联系 |
| **抽象拓展** | 迁移应用与创新思考 | 能将"坐忘"七阶转化为现代人的静心练习方案；能比较司马承祯与禅宗修行方法的异同；能设计个人的"坐忘"实践 |

### 三、PBL驱动问题

**核心驱动问题**：在纷扰的世界中，如何通过"坐忘"达到内心的宁静？司马承祯的七阶修行法对我们设计个人静心练习有什么启发？

**子问题链**：
1. "坐忘"出自庄子，司马承祯如何将其系统化？
2. 七阶修行法的每一阶具体要做什么？
3. "敬信"为什么是第一阶？不信道的人能修炼吗？
4. "得道"是怎样的状态？

### 四、分阶段学习任务

#### 阶段一：初识坐忘（第1周）—— 前结构→单点结构

**学习内容**：
- 阅读坐忘论全文
- 了解司马承祯的生平与思想

**任务清单**：
- [ ] 通读坐忘论全文
- [ ] 写出"坐忘"对你意味着什么（200字）

**作品产出**：
- 抄经作品：工整抄写坐忘论全文
- 初读笔记：《坐忘论初印象》

#### 阶段二：七阶精研（第2周）—— 单点结构→多点结构

**学习内容**：
- 精读七阶的每一阶
- 理解每一阶的具体修行方法，整理七阶之间的关系

**任务清单**：
- [ ] 制作"坐忘七阶详解表"（每阶的核心要求、修行方法、常见障碍）
- [ ] 画出七阶递进关系图

**作品产出**：
- 思维导图：《坐忘七阶修行指南》
- 学习体会：《坐忘七阶的修行密码》（800-1500字）

#### 阶段三：思想关联与静心实践（第3周）—— 多点结构→抽象拓展

**学习内容**：
- 将坐忘论与清静经、道德经进行比较
- 设计基于坐忘论的个人静心方案
- 将坐忘智慧应用于日常生活

**任务清单**：
- [ ] 写出坐忘论与清静经的修行方法比较（500字）
- [ ] 设计一份"21天坐忘静心方案"（参考七阶递进）
- [ ] 用坐忘七阶的框架分析清静经的修行路径

**作品产出**：
- 比较分析：《坐忘与清静：道教静修的两条路径》（1000字）
- 实践方案：《我的21天坐忘静心计划》
- 交流文章：《坐忘论给现代人的静心启示》（1000-1500字）
