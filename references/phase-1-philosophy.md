# 第一阶段：哲学根基 —— 道德经 + 庄子

> 建立"道"的世界观与思维方式（约5-6周 · 道德经5,162字 + 庄子内篇14,000字）

## 第一章：道德经

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《道德真经》 |
| **作者** | 老子（李耳），春秋时期楚国人 |
| **年代** | 约公元前6世纪（成书约战国初期） |
| **篇幅** | 81章，约5000字 |
| **结构** | 道经（1-37章）+ 德经（38-81章） |
| **核心思想** | 道法自然、无为而治、上善若水、反者道之动 |
| **历史地位** | 道家哲学根本经典，与《易经》《论语》并称中国三大思想源泉 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能听过"道可道非常道"，但不理解其含义；对老子的印象仅限于神话传说 |
| **单点结构** | 识别一个核心概念 | 能解释"道"的基本含义——万物的本源与规律；或能解释"无为"的字面意思 |
| **多点结构** | 列举多个关键概念 | 能说出道、德、无为、自然、水、柔弱、不争等核心概念；能区分道经与德经的不同侧重 |
| **关联结构** | 理解概念间的内在联系 | 能解释"道"与"德"的关系（道为体、德为用）；能理解"无为"并非无所作为而是不妄为；能将"反者道之动"与"柔弱胜刚强"联系起来 |
| **抽象拓展** | 迁移应用与创新思考 | 能将"无为"思想应用于现代管理、人际关系；能比较老子与其他哲学家（如孔子、释迦牟尼）的思想异同；能从道德经中提炼出个人的人生哲学 |

### 三、PBL驱动问题

**核心驱动问题**：在一个人人都在"做加法"的时代，《道德经》的"减法智慧"能给我们的生活带来什么启示？

**子问题链**：
1. "道"究竟是什么？为什么老子说它"不可言说"？
2. "无为而治"在今天还有意义吗？
3. 为什么老子反复强调"柔弱""不争""处下"？
4. 道德经81章中，哪一章最触动你？为什么？

### 四、分阶段学习任务

#### 阶段一：初识道德经（第1周）—— 前结构→单点结构

**学习内容**：
- 通读道德经全文（建议选择权威注释本，如陈鼓应《老子今注今译》）
- 不求甚解，标记印象深刻的句子

**任务清单**：
- [ ] 通读全文，标记20个以上触动你的句子
- [ ] 用自己的话写出"道是什么"（100字以内）

**作品产出**：
- 《初读道德经——我的20个句子》笔记（附简短感想）

#### 阶段二：概念梳理（第2周）—— 单点结构→多点结构

**学习内容**：
- 精读核心章节（建议重点：1、8、11、16、22、25、33、37、40、42、48、63、76、78、81章）
- 整理核心概念清单

**任务清单**：
- [ ] 制作"道德经核心概念表"（至少15个概念，含原文出处、白话解释、个人理解）
- [ ] 尝试背诵5个以上经典句子

**作品产出**：
- 思维导图：《道德经核心概念图谱》
- 学习体会文章：《我理解的道德经"关键词"》（800-1500字）

#### 阶段三：融会贯通（第3周）—— 多点结构→关联结构

**学习内容**：
- 研读道德经的整体结构与思想脉络
- 理解"道—德—万物"的生成论
- 把握"反者道之动，弱者道之用"的核心逻辑

**任务清单**：
- [ ] 绘制道德经思想体系图（道→德→万物→修身→治国的逻辑链）
- [ ] 选择一个现实问题，用道德经的思想进行分析

**作品产出**：
- 知识图谱：《道德经思想体系全景图》
- 实践记录：《用道德经的眼光看______》（选择一个社会现象或个人困惑）

#### 阶段四：知行合一（第4周）—— 关联结构→抽象拓展

**学习内容**：
- 将道德经智慧与个人生活、工作结合
- 思考道德经在现代社会的价值

**任务清单**：
- [ ] 选择道德经中的一个智慧（如"上善若水""知足不辱"），制定21天实践计划
- [ ] 记录实践过程中的心得与困惑

**作品产出**：
- 实践报告：《21天道德经生活实验》（含每日记录要点）
- 交流文章：《写给朋友的道德经推荐信》（800-1500字）

### 五、推荐参考资源

**入门读物**：

## 第二章：庄子

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《南华真经》 |
| **作者** | 庄子（庄周），战国时期宋国蒙人 |
| **年代** | 约公元前4世纪 |
| **篇幅** | 33篇（内篇7 + 外篇15 + 杂篇11），约6.5万字 |
| **核心思想** | 逍遥游、齐物论、养生主、无用之用、物化 |
| **历史地位** | 道家哲学双壁之一（老庄并称），中国文学与哲学的巅峰之作 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能听过"庄周梦蝶""庖丁解牛"的故事，但不知出处；对庄子的印象是"消极避世" |
| **单点结构** | 识别一个核心概念 | 能解释"逍遥"的含义——不依赖外物的精神自由；或能讲出"齐物论"的基本意思 |
| **多点结构** | 列举多个关键概念 | 能说出逍遥、齐物、养生、无用、物化、坐忘、心斋等核心概念；能区分内篇、外篇、杂篇的不同 |
| **关联结构** | 理解概念间的内在联系 | 能理解"齐物论"是"逍遥游"的哲学基础（万物齐一才能真正逍遥）；能将"无用之用"与"养生主"联系起来 |
| **抽象拓展** | 迁移应用与创新思考 | 能用庄子的"齐物"视角看待不同文化和观点；能将"逍遥"精神应用于应对现代焦虑；能比较庄子与西方存在主义的异同 |

### 三、PBL驱动问题

**核心驱动问题**：在一个充满比较和竞争的社会里，庄子的"逍遥"究竟是一种逃避，还是一种更高维度的自由？

**子问题链**：
1. 鲲鹏与蜩鸠的故事，庄子想告诉我们什么？
2. "子非鱼，安知鱼之乐"——我们真的能理解他人吗？
3. 庄子为什么说"无用"才是最大的"用"？
4. "庄周梦蝶"提出了怎样的哲学问题？

### 四、分阶段学习任务

#### 阶段一：故事先行（第1周）—— 前结构→单点结构

**学习内容**：
- 阅读庄子中最著名的寓言故事（鲲鹏、庖丁解牛、庄周梦蝶、濠梁之辩、无用之树等）
- 了解庄子的生平与时代背景

**任务清单**：
- [ ] 阅读至少10个庄子寓言故事
- [ ] 选择最打动你的一个故事，用自己的话复述（500字）

**作品产出**：
- 《我最喜欢的庄子故事》图文笔记

#### 阶段二：精读内篇（第2周）—— 单点结构→多点结构

**学习内容**：
- 精读内篇7篇（逍遥游、齐物论、养生主、人间世、德充符、大宗师、应帝王）
- 理解每篇的核心论点

**任务清单**：
- [ ] 精读内篇7篇，每篇写200字摘要
- [ ] 制作"庄子核心概念表"（至少10个概念）

**作品产出**：
- 思维导图：《庄子内篇思想导图》
- 学习体会：《庄子教会我的第一件事》（800-1500字）

#### 阶段三：思想融通（第3周）—— 多点结构→关联结构

**学习内容**：
- 将内篇思想与道德经进行比较
- 理解庄子对老子思想的继承与发展
- 把握"齐物—逍遥—养生"的思想链条

**任务清单**：
- [ ] 写出"老子与庄子的异同"分析（500字）
- [ ] 用庄子的视角分析一个现代人的精神困境

**作品产出**：
- 知识图谱：《老庄思想比较图》
- 交流文章：《庄子眼中的"自由"》（1000-1500字）

#### 阶段四：逍遥实践（第4周）—— 关联结构→抽象拓展

**学习内容**：
- 将庄子的逍遥精神与日常生活结合
- 探索"无用之用"在现代生活中的应用

**任务清单**：
- [ ] 尝试"无用时光"实验：每周留出2小时做"无用"之事，记录感受
- [ ] 用庄子的思想写一封信给"焦虑的自己"

**作品产出**：
- 实践报告：《我的"无用时光"实验记录》
- 创意写作：《写给焦虑的自己的庄子式信件》（1000字）
