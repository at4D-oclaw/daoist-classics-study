## 第十章：太上感应篇

### 一、典籍概览

| 项目 | 内容 |
|------|------|
| **全称** | 《太上感应篇》 |
| **作者** | 传为太上老君所说，实为宋代道流所编 |
| **年代** | 约成书于北宋末年 |
| **篇幅** | 约1200字 |
| **核心思想** | 善恶报应、祸福无门、惟人自召、积善成仙 |
| **历史地位** | 道教"善书之首"，与《文昌帝君阴骘文》《关圣帝君觉世真经》并称三大善书 |

### 二、SOLO五层知识框架

| 认知层级 | 学习目标 | 具体表现 |
|----------|----------|----------|
| **前结构** | 零基础状态 | 可能听过"善有善报，恶有恶报"，但不知道它出自道教经典；对"善书"没有概念 |
| **单点结构** | 识别一个核心概念 | 能解释"感应"的基本含义——善恶行为会感召相应的报应；或能说出"祸福无门，惟人自召"的含义 |
| **多点结构** | 列举多个关键概念 | 能说出善恶、感应、报应、积善、改过、修身等核心概念；能列举太上感应篇中的主要善行和恶行 |
| **关联结构** | 理解概念间的内在联系 | 能理解"感应"不是简单的因果报应而是天人相应的宇宙法则；能将太上感应篇与道德经的"天道"观联系 |
| **抽象拓展** | 迁移应用与创新思考 | 能理解道教善书的社会教化功能；能比较不同宗教的善恶报应观；能将"积善"理念应用于个人道德修养 |

### 三、PBL驱动问题

**核心驱动问题**："善有善报，恶有恶报"是简单的因果报应，还是一种更深层的宇宙法则？太上感应篇的"积善"理念如何指导我们的日常生活？

**子问题链**：
1. 为什么太上感应篇被称为"善书之首"？
2. "祸福无门，惟人自召"——人真的是自己命运的主人吗？
3. 太上感应篇列举的善行和恶行，哪些最触动你？
4. 道教的"善"与其他宗教的"善"有什么不同？
